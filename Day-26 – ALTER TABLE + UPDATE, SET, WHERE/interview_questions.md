# Day 26 – Interview Questions

## 1. What is ALTER TABLE?
ALTER TABLE is used to modify the structure of an existing table.

---

## 2. How do you add a column to a table?
ALTER TABLE table_name ADD column_name datatype;

---

## 3. How do you delete a column?
ALTER TABLE table_name DROP COLUMN column_name;

---

## 4. What is UPDATE in SQL?
UPDATE is used to modify existing records in a table.

---

## 5. What is the role of SET in UPDATE?
SET assigns new values to columns.

---

## 6. Why is WHERE important in UPDATE?
WHERE ensures only specific rows are updated.

---

## 7. What happens if WHERE is not used in UPDATE?
All rows in the table will be updated.

---

## 8. Can we update multiple columns at once?
Yes, using comma-separated values in SET.

---

## 9. Difference between ALTER and UPDATE?
ALTER changes table structure.
UPDATE changes data.

---

## 10. Give a real-world example of UPDATE.
Updating salary of employees or correcting user details.
