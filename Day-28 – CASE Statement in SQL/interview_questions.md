# Day 28 – Interview Questions

## 1. What is CASE in SQL?
CASE is used to implement conditional logic like IF-ELSE.

---

## 2. Types of CASE?
Simple CASE and Searched CASE.

---

## 3. Difference between Simple and Searched CASE?
Simple CASE compares values.
Searched CASE uses conditions.

---

## 4. Can CASE be used in UPDATE?
Yes, it can be used to update values conditionally.

---

## 5. What happens if ELSE is not used?
Returns NULL if no condition matches.

---

## 6. Can CASE be used with aggregation?
Yes, for conditional counting or summing.

---

## 7. Is CASE a function?
No, it is a SQL expression.

---

## 8. Where can CASE be used?
SELECT, UPDATE, ORDER BY, GROUP BY.

---

## 9. Give real-world use of CASE.
Categorizing users, grading systems, dynamic calculations.

---

## 10. Common mistake in CASE?
Forgetting END or wrong condition order.
