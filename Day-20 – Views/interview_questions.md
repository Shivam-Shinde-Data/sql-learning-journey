# Day 20 – Interview Questions (Views)

## 1. What is a view?
A virtual table based on a query.

## 2. Does a view store data?
No, it stores only the query.

## 3. Why use views?
Security, simplicity, reusability.

## 4. Difference between table and view?
Table stores data, view does not.

## 5. Can we update a view?
Yes, only simple views.

## 6. What is a complex view?
View with joins, group by, functions.

## 7. How to delete a view?
```sql
DROP VIEW view_name;
```
## 8. Can views improve performance?
Yes, by simplifying queries (but not always faster).

## 9. What is data abstraction in views?
Hiding complex logic from users.

## 10. Can we use JOIN in views?
Yes, in complex views.
