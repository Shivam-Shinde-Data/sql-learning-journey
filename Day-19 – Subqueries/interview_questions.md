# Day 19 – Interview Questions (Subqueries)

## 1. What is a subquery?
A query inside another query.

## 2. How does a subquery execute?
Inner query runs first, then outer query.

## 3. Types of subqueries?
Single row, multiple row, correlated.

## 4. What is a correlated subquery?
A subquery that depends on outer query.

## 5. Difference between IN and EXISTS?
IN compares values, EXISTS checks presence.

## 6. Where can subqueries be used?
SELECT, WHERE, FROM.

## 7. What is a derived table?
Subquery used in FROM clause.

## 8. Difference between ANY and ALL?
ANY → at least one match  
ALL → must satisfy all values

## 9. Are subqueries slow?
Can be slower than joins in large datasets.

## 10. Alternative to subqueries?
JOIN operations
