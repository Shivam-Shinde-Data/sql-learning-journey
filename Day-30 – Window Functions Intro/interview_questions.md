# Day 30 – Window Functions Interview Questions

## 1. What are window functions?
Functions that perform calculations across a set of rows without reducing rows.

---

## 2. Difference between GROUP BY and Window Functions?
GROUP BY collapses rows, window functions do not.

---

## 3. What is OVER() clause?
Defines the window (set of rows) for calculation.

---

## 4. What is PARTITION BY?
Divides data into groups for window functions.

---

## 5. What is ORDER BY in window functions?
Defines order inside partition.

---

## 6. Can we use aggregate functions as window functions?
Yes (SUM, AVG, COUNT, etc.)

---

## 7. Give real-world use case?
Running totals, leaderboards, analytics.

---

## 8. Does window function reduce rows?
No.

---

## 9. Can we use multiple window functions in one query?
Yes.

---

## 10. Is window function important for Data Engineering?
Yes, heavily used in analytics pipelines.
